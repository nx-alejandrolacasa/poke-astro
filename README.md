# Pokémon Pokédex

A modern, responsive Pokédex web application to explore the wonderful world of Pokémon. Browse through all species, discover their stats, abilities, evolutions, and type effectiveness.

*Gotta catch 'em all!*

## Features

- **Complete Pokédex** - Browse all 1000+ Pokémon species with infinite scroll
- **Detailed Pokémon Pages** - View stats, abilities, type effectiveness, and evolution chains
- **Type Browser** - Explore Pokémon by their elemental types
- **Generation Browser** - Discover Pokémon organized by generation and region
- **Smart Search** - Quickly find any Pokémon by name
- **Dark Mode** - Easy on the eyes for those late-night Pokémon research sessions
- **Bilingual Support** - Available in English and Spanish
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile

## Tech Stack

- **[Astro](https://astro.build)** - Static-first web framework with SSR capabilities
- **[React](https://react.dev)** - For interactive components
- **[Tailwind CSS](https://tailwindcss.com)** - Utility-first styling
- **[TypeScript](https://www.typescriptlang.org)** - Type-safe development
- **[Vercel](https://vercel.com)** - Serverless deployment

## Getting Started

### Prerequisites

- Node.js 20.x or higher
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/nx-alejandrolacasa/poke-astro.git
cd poke-astro

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Start local dev server at `localhost:3000`  |
| `npm run build`   | Build production site to `./dist/`          |
| `npm run preview` | Preview build locally before deploying      |
